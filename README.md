# bsc-nft-scraper

## Installation

```bash
npm install web3
```

Run local full node
```bash
./geth_linux --config ./config.toml --datadir ./node  --cache 18000 --rpc.allow-unprotected-txs --txlookuplimit 0 --ipcpath ~/.bsc --ws
```
